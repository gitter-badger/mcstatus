# mcstatus
minecraft status checker.

纯 C++ 编写，用了 boost 库，欢迎点星。
![screenshot1](https://raw.githubusercontent.com/jingjinghack/jingjinghack/master/screenshot.png)

# How to build
On Debian/Ubuntu:
```
# apt-get install libboost-dev
$ mkdir build; cd build
$ cmake ..
$ make
```
On Arch Linux: 
```
# pacman -S boost
$ mkdir build; cd build
$ cmake ..
$ make
```
